Security Center on Mobile Apps

  • Category: Mobile App Enhancement, Security Feature
  • Company: Lowe's
  • Deployment: 20 October, 2022
  • URL: www.lowes.com

Description

Empowering users with enhanced security controls on-the-go!

Users encountered challenges when attempting to manage their security settings exclusively through desktop or mobile web experiences, leading to:

  • Friction in enabling MFA: Users Users seeking to enable MFA were required to navigate to the desktop website, disrupting the user experience and introducing unnecessary complexity.

    • Customer Benefits

    The introduction of the Security Center on Mobile Apps resolves these challenges:

  • Reduced friction for mobile app users: Users can now conveniently enable MFA and manage their security settings directly within the Lowe's Mobile App, eliminating the need to switch between platforms for security management.

    • Impact

    The Security Center and MFA offer numerous benefits for Lowe's customers:

  • Decreased fraud online chargebacks: As more customers embrace MFA, fraudulent activities and associated chargebacks were expected to decline, enhancing overall security and trust.
  • Enhanced protection against account takeovers: MFA serves as an additional layer of defense, mitigating the risk of unauthorized access to users' accounts and reducing liabilities for both customers and Lowe's.
