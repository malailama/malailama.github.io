Sign In Without Password Using Email OTP

  • Category: User Experience, Security Enhancement
  • Company: Lowe's
  • Deployment: 27 September, 2022
  • URL: www.lowes.com

Description

Streamlining sign-in with enhanced security!

Previously, users faced challenges related to password fatigue and friction at sign-in:

  • Password fatigue: Users grappled with the burden of remembering numerous passwords, leading to frustration and inefficiency.
  • Friction at sign-in: Forgotten passwords necessitated a cumbersome password reset journey, deterring users from seamless access to their accounts.

    • Customer Benefits

    The Sign In without Password feature offers a streamlined solution to these challenges:

  • Flexibility and security: Users can now opt for Email OTP authentication, bypassing the need for password memorization while ensuring account security.
  • Reduced friction: By eliminating the need for password resets, the sign-in process becomes more efficient and user-friendly.

    • Impact

    The implementation of Sign In with OTP has resulted in significant improvements:

  • Success percentage for Login with OTP: 72.2%
  • Over 50,000 users have successfully signed in using OTP in first week of launch.
  • 23.5% of users visiting the Login page have opted for 'Sign In using OTP,' indicating high user interest and adoption.
